Support

RUPN Server is an open-source Docker image for running a single personal server instance. For help, use the channels below.

GitHub Issues

Report bugs, deployment issues, and documentation problems at github.com/makamekm/ruvpn-server/issues.

Before opening an issue

  • Check the container logs for the RUPN server started marker.
  • Confirm that Docker can pull makame/rupn-server:latest.
  • Do not publish RUPN_CONNECT_URI or the volume contents.

Useful command

docker logs rupn-server
