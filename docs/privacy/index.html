Privacy Policy

This Privacy Policy describes how RUPN Server handles information. RUPN Server is open-source infrastructure that users deploy on their own servers.

Information collected by this website

This static website does not provide accounts, analytics, forms, tracking pixels, advertising SDKs, or payment flows.

Information processed by your server

When you run the Docker container, connection material is generated and stored locally in your Docker volume. The project maintainers do not receive your connection links, keys, logs, traffic, IP addresses, or usage data.

Connection links and keys

RUPN_CONNECT_URI contains raw connection material and should be kept private. RUPN_CONNECT_JWT is the user-facing signed token expected by the client app.

Third-party services

The project is hosted on GitHub Pages and the Docker image is distributed through Docker Hub. Their respective privacy policies apply to those platforms.

Contact

For privacy questions, open an issue at GitHub Issues.
